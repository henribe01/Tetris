# Tetris
A simple Tetris game written in Python using Pygame. 
## Controls
- Left: Move left
- Right: Move right
- Down: Move down
- Q: Rotate left
- E: Rotate right
- Space: Hard drop

## Installation
1. Clone the repository
2. Install the required packages using `pip install -r requirements.txt`
3. Run the game using `python tetris.py`
4. Enjoy!
